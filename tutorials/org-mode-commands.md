Org Mode Brief Guide
====================

## Header
```
#+AUTHOR: name here
#+TITLE: title here
#+OPTIONS: ^:{}
#+HTML_DOCTYPE: html5
# +INFOJS_OPT: view:overview toc:t
```

## Major Headings

```
* h1
```

## Minor Headings
```
** h2
```
Toggle headings, using <Tab>

## Links

### insert link

``` C-c C-l ```

OR, like this:

```
[[http://work.arcolife.in/][Archit's homepage]].
```

### Open Links

```C-c C-o```

## Quotes

```
#+BEGIN_QUOTE
your quote
#+END_QUOTE
```

## Examples (<pre>)

```
#+BEGIN_EXAMPLE
example
#+END_EXAMPLE
```

## Styling

### Points

```
- xyz
- abc 
```

### Bold

```
=oh yeah=
```

### Para points

```
Therefore:

- =oh= :: blabla
       	  blabla
- =yea= :: hmm
  	   hmm

```
